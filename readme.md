## Contoh penggunaan js ke dalam html

* Seleksi element menggunakan query selector https://www.w3schools.com/JSREF/met_element_queryselector.asp
* Gunakan innerText atau textContent untuk memasukkan nilai js kedalam html sesuai elemnt yang diseleksi https://www.w3schools.com/JSREF/prop_node_innertext.asp
* Gunakan fungsi untuk update waktu seperti setInterval atau setTimeout https://www.w3schools.com/JSREF/met_win_settimeout.asp
